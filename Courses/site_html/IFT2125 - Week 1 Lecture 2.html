Part 1 - Asymptotic Complexity

Il faut retenir les definitions de: - Big O: Borne superieure (<=) : il existe un c pour lequel tous n>n_0 -> f(n) <= c * g(n) - Little O: Borne superieure (<) : pour tout c et n > n_0 -> f(n) <= c * g(n) - Big Omega: Borne inferieure (>=) : g \elem Big-O(f) - Little omega: Borne inferieure (>) : g \elem little-o(f) - Big Theta: Egalite des bornes (=) : f \elem Big-O(f) && g \elem Big-O(f)
