MAT1720 - Section - Variables Aléatoires Discrètes

Overview

On peut modéliser plusieurs types de variables aléatoires discrètes: - Loi Binomiale - Loi Géométrique - Loi Hypergéométrique - Loi Binomiale Négative - Loi de Poisson - Processus de Poisson

Il est à noter que la ditribution de chacune de ces variables peut se trouver avec un peu d'intuition: on soit se demander comment piger.
