Part 2 - Simplification Map de Karnaugh et Quine-McKlusey

Pour simplifier les fonctions, on peut utiliser 2 méthodes: - table de Karnaugh: code grey + regroupement en puissance de 2 - Quine-McKluskey: regroupement des minterms et don't care par groupe de bits; comparer les différences entre les bits
